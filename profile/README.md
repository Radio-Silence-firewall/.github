# Radio Silence — Portable Simple Firewall to Block Individual Apps

## About Radio Silence

Radio Silence provides a complete simple firewall environment for users who want to block internet access for specific applications. Radio Silence combines per‑app blocking, list management, and real‑time monitoring. Radio Silence enables users to prevent apps from phoning home, sending analytics, or using network without permission. Radio Silence runs on desktop platforms. Radio Silence is trusted by privacy‑conscious users, developers, security professionals, and anyone who wants simple app blocking.

## Deep Dive — Simple Firewall Notes

Per‑app blocking: block any app from accessing internet, no configuration
App list: add any app by dragging into window, remove with one click
Real‑time monitoring: see which apps are trying to connect (live)
No rules: no IP addresses, ports, protocols — just app names
Permanent: blocked apps stay blocked until you remove them
